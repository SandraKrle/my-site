My work at Svenska Spel

At Svenska Spel I work as a CMS frontend developer. The CMS is an inhouse built web app to manage the content on Svenska Spels's web site and game products. I am one of three frontend developers in a team of eight persons including backend, UI/UX and a Product Owner.

The frontend stack is React + Redux + Node

Developing and maintaining the CMS includes:

  • Developing new features (e.g. new types of widgets, banners, time scheduling tools for the content, A/B testing support)
  • Maintenance of existing features (bug fixing, rewriting legacy code from Backbone to React, performance optimization, updating of tests)
  • Writing unit tests with Enzyme, functional (UI) and integration (API) tests with Mocha and webdriver.io.
  • Gathering input from the CMS users in terms of usability and improvements and translating it into requirements
  • Close coordination with the back-end developers
